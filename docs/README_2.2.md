# T2灰度映射法 — 技术说明

## 一句话
**用一张T2图像的灰度值，直接线性映射出ADC真值图，再前向生成DWI模拟数据。**

## 数据来源
- **模板**：MNI152 T2加权标准模板（ICBM 2009a），3D → 提取2D切片
- **脑mask**：同一空间的T1脑mask，用于去除颅骨/背景

## 核心公式

```
ADC(x,y) = 2.4×10⁻³ · I_T2(x,y) + 0.6×10⁻³
```

| 组织 | T2亮度 | ADC值 |
|------|--------|-------|
| 白质(WM) | 暗 | 0.6~0.7×10⁻³ |
| 灰质(GM) | 中 | 0.9~1.2×10⁻³ |
| 脑脊液(CSF) | 亮 | 2.5~3.0×10⁻³ |

**关键**：T2亮度与ADC**正相关**——CSF最亮/扩散最强，WM最暗/扩散受限。

## S0和K的推导

S0和K也从T2灰度**连续映射**（非分段赋值）：
- **S0 = 0.85 + 0.35 × I_T2**（CSF亮→S0高）
- **K = 1.0 - 0.95 × I_T2**（WM暗→K高）

每个像素有独立的参数值，保留组织内部的自然纹理。

## 前向DWI生成（DKI模型）

```
S(b) = S₀ · exp(-b·D + ⅙·b²·D²·K)
```

b = [0, 500, 1000, 1500, 2000] → 每个样本5张DWI

加Rician噪声模拟真实MRI噪声分布。

## 与路径2.3（组织分割法）的区别

| | 本方案（2.2灰度映射） | 2.3组织分割 |
|--|----------------------|------------|
| 输入 | 单张T2灰度图 | GM/WM/CSF概率图 |
| 需要分割 | ❌ 不需要 | ✅ 需要硬分割 |
| 参数空间 | 连续渐变、像素级不同 | 分段均匀、片内相同 |
| 解剖细节 | T2自带纹理，真实度高 | 需额外加随机扰动 |
| 靠什么解释给老师 | "T2信号与ADC天然正相关" | "每种组织赋予不同D/K参数" |

## 生成策略

10个样本 = 10种不同轴向切片位置 + 5种噪声水平 + 不同随机扰动种子
